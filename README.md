### Projects

**[Textile Industries in Lyon, 1843–1875](https://github.com/AlessandroVerduna/thesis-textile-lyon)**  
Master's thesis. Serial quantitative analysis of 1,823 company acts deposited at the Tribunal de Commerce de Lyon. Archival research, Python, Jupyter. Conducted within the ANR-funded DIRIVA programme at LARHRA.

**[Società Storica Pinerolese — AWS Infrastructure](https://github.com/AlessandroVerduna/societastoricapinerolese-website)**  
End-to-end serverless website infrastructure. S3, CloudFront with OAC, ACM, Route 53. All defined as code with Terraform.

**[News Aggregator — RSS + AI Summary](https://github.com/AlessandroVerduna/progetto-feed-rss)**  
CLI tool in Python. RSS feed fetching, keyword search, SQLite persistence, AI summaries via Groq, PDF export with ReportLab.
